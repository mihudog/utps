IMRT计划设计步骤

添加计划&射束组

  1. 主模块切换到计划模块（子模块也切换到计划列）
  2. 点击添加计划
  3. 选择机器、辐射类型、技术类型、算法
  4. 设置处方参数：如果关联了计划申请单，BG中会自动读取处方单上的数值
  5. 点击确定

添加射束

  1. 点击添加射束
  2. 等中心选择POI或ROI
    1. 这里的等中心特指计划中心
    2. 系统将计划中心和POI分离关联
    3. 等中心默认名称是“射束组名称+_1”
    4. 等中心可以通过POI或ROI获取坐标，如果改变POI坐标，等中心坐标不会关联变化
    5. 射束列表中可以修改等中心的坐标
  3. 设定照射模式和剂量率
  4. 设置各项机械参数
  5. 点击OK

    注：新增射束在BEV视图中显示闭合的透明MLC样式，如下图

添加优化约束

  1. 切换到优化约束
  2. 点击添加
  3. 可选择的优化函数类型有
    1. 最小剂量
    2. 最大剂量
    3. 平均剂量
    4. 最小DVH
    5. 最大DVH
    6. 最小EUD
    7. 最大EUD
    8. 优化辅助环
    9. 单值约束
      对于需要避开的ROI，可以添加单值约束（Single Value）剂量为0的优化约束条件，可实现Avoidance功能。
      • 仅适用于射束路径上的危及器官在靶区之前的场景。
      • IMRT类型完全不会在Avoidance区域开野，uArc计划只在必要情况下才会在Avoidance区域滑过。
  4. 权重
    1. 数值范围：1-10000
    2. 强约束1=弱约束100
  5. 优化条件前面的勾选框选中表示该条件参与优化，不勾选表示不参与优化

优化设置与开始优化

  1. 开始优化前可点击优化设置打开优化设置窗口
    1. 迭代次数越高，优化所需时长越长
    2. 如果治疗技术为uARC，系统提供两种生成控制点的方法
      1） FMO（Fluence Map Optimization）
      2） SPO （Stochastic Platform Optimization)

    3. 如果治疗技术为dIMRT，系统支持调整最大子野个数和迭代次数。
    4. 如果治疗技术为sIMRT，还支持调整最小子野面积、最小子野MU、和最大子野个数。
  2. 完成优化设置后，点击开始优化开始迭代优化
    1. 每轮优化结束后会自动进行最终剂量的计算，不需要再点计算！

评估计划

等剂量线

在剂量阶梯上方打开等剂量设置
  1. 系统提供了2种剂量分级模式：相对模式和绝对模式
    相对与绝对等剂量线由100%剂量设定来转换，通常采用处方目标剂量作为100%等剂量线。
  2. 系统提供3种显示模式：Line、Band、Colorwash
    Band和Colorwash模式下可调节剂量填充显示的透明度。

  3. 支持等剂量线模板的选择、编辑、另存、删除以及设置默认等剂量线。

DVH和ROI统计

DVH视图

DVH视图具备以下常用功能：
  1. 点击1处可关闭可显示ROI的DVH曲线，支持全显示和全隐藏
  2. 点击2处可下拉选择ROI列表的排序方式，支持顺序、倒序、靶区优先、优化条件优先和ROI列表等多种排序方式
  3. 光标移到曲线上可测量该ROI的在该处的剂量和体积信息
  4. 滚动鼠标滚轮可缩放DVH视图，按住鼠标左右键可移动视图，点击重置可将DVH视图恢复到默认状态
  5. 点击5处眼睛可关闭或打开约束条件小箭头
  6. 点击刷新，可重新统计DVH数据，对于新增或编辑过的ROI，可不用重新计算剂量而得到更新后的DVH数据
  7. 支持将DVH数据和ROI统计数据导出未.json格式

ROI剂量统计

  1. 点击信息栏的ROI统计-剂量信息
  2. 列表中的结构和DVH上显示的ROI曲线对应。在DVH右侧的ROI列表中点选结构，相应ROI的剂量统计信息会同步出现在统计列表。
  3. 剂量信息统计表可设置D#%（百分比体积的剂量），D#cc（绝对体积的剂量），V#Gy（剂量体积）
  4. +和-可以增加和减少统计列

最大剂量点

  1. 在界面顶部工具栏选择最大最小剂量
  2. 选择最大剂量
  3. 选择皮肤（skin）
  4. 点击跳转CT图上即会显示最大剂量点位置

剂量剖面

  1. 显示剂量剖面窗口后，点击剂量剖面按钮
  2. 在MPR平面上点一下确定剖线的起始位置，然后拖动光标到另一点再点一下生成剖线，期间会出现角度指示

    MPR三个平面上分别做任意角度剖线，剂量剖面窗口对应显示剂量与距离的分布关系

  3. 对每条剖线统计多条剂量线，对应射束组的剂量和每个子野贡献的剂量，光标悬停在线上有相应提示

目标表和模版

  1. 目标表支持的目标类型分为
    1. 体积剂量D#
    2. 剂量体积V#
    3. 最大剂量
    4. 最小剂量
    5. 平均剂量
  2. 其中体积类型分为相对体积和绝对体积
  3. 添加目标表后可以从下图第7步看实际值是否达到目标值
  4. 目标表支持模版
  5. 添加目标表和保存模版的步骤如下图所示

根据目标值调参

  1. 通过初步评估DVH和ROI剂量统计信息，调整优化条件
  2. 优化条件的目标值可为调整策略提供参考
    1. 目标值采用的是科学计数法
    2. 目标值越大，说明该条件对应的当前实际值距离设定的目标较远，该条件设定的太“苛刻”
    3. 目标值越小，说明对应的当前实际值距设定的目标较近，该条件设定的恰当
    4. 目标值为0，说明当前值已达到了目标，该条件设定的“宽松”，没有起作用
  3. 根据目标值调整优化约束后，点击继续优化

剂量阈值分割

  1. 在ROI列表栏点剂量分割
  2. 填入积分剂量数值
    -注意不要不要勾选剂量区间
  3. 设定ROI名称和角色
  4. 点击OK

  5. 到勾画界面继续处理生成的ROI

继续优化

在优化条件中添加新的结构，继续优化

剂量归一

  1. 在顶部工具栏选择剂量归一
  2. 三种归一方式
    1. 按处方归一 ：归一到当前处方
    2. 按ROI 归一 ：按ROI的平均值归一 或 按ROI的体积剂量归一
    3. 按POI归一
  3. 点击OK

  4. 归一改变的是所有子野的MU数，反映到TPS输出的结果上就是单次量的改变。归一比例最大不可超过当前值的10%，否则可能报错MU与机架冲突，导致计划无法被批准执行
  5. 若改变处方的分次量，应该重新做优化
