导入DICOM数据

在 DICOM 导入模块中，DICOM数据来源有三种，分别为本地磁盘，服务器，PACS。

从本地磁盘导入患者数据

左侧导航栏选择DICOM导入，打开如下页面
  1. 数据来源选择本地磁盘；
  2. 浏览需要导入数据的文件夹路径；
  3. 选中要导入的文件夹，点击上传；
  4. 勾选患者信息；
  5. 勾选数据；
    点击图像序列后的眼睛图标可以预览图像。
  6. 导入。
    完成后弹窗提示导入是否成功。

从服务器导入患者数据

  1. 点击DICOM导入，进入 DICOM 导入页面。
  2. 在数据来源下拉列表中选择服务器；
  3. 勾选需要导入的数据；
  4. 导入。
    Note: 导入时可以通过患者 ID 进行搜索和刷新搜索。

从PACS系统中导入

需要提前配置好PACS系统的 AE 节点，并获取该PACS的Query权限（需要PACS厂家或院方管理部门配合）。

在数据来源下拉列表中选择 PACS，在服务器下拉列表中选择PACS服务器。在弹窗中查找并移动序列，具体操作如下。
  1. 通过筛选条件查询患者；
  2. 患者列表中选择患者；
  3. 序列列表中选择序列；
  4. 移动序列。
